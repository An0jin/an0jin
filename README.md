# 안영진 (anyoungjin)

## 자기소개
안녕하세요! 저는 중학교 3학년 때부터 다양한 프로그래밍 언어를 공부해온 안영진입니다. 현재는 파이썬을 활용한 데이터 분석과 웹 애플리케이션 개발에 집중하고 있으며, 꾸준히 새로운 기술을 배우고 있습니다.
